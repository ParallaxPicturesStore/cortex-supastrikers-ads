# Cortex — SupaStrikers Ad Assets

Public CDN for Meta ad creative used in the SupaStrikers coach recruitment campaign. Built by Cortex / The Noisy Marketers.

## Structure

```
ads/                  Single-image ad creatives (1:1 1080×1080 PNGs)
carousel-cards/       Action-shot carousel cards (1:1 1080×1080 PNGs)
```

## Usage

Images are served from `https://raw.githubusercontent.com/ParallaxPicturesStore/cortex-supastrikers-ads/main/<path>`. Meta Ads Manager fetches these as `image_url` values during creative creation.
